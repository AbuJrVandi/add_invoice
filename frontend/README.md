# Invoice Management Frontend (React + Vite)

## Requirements

- Node.js 20+
- npm

## Setup

1. Install dependencies:
   - `npm install`
2. Configure env:
   - `cp .env.example .env`
3. Run dev server:
   - `npm run dev`

## Notes

- Ensure backend API is running on `http://localhost:8000`.
- Auth token is stored in localStorage and sent as Bearer token for protected routes.
